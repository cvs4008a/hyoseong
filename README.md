# Hyoseong - Partnership Inquiry Form

A professional, high-performance web page featuring a partnership inquiry form integrated with [Formspree](https://formspree.io/).

## 🚀 Features
- **Modern UI/UX**: Built with CSS Baseline features like `oklch` colors, Container Queries, and subtle textures.
- **Formspree Integration**: Ready-to-use backend for form submissions without needing a server.
- **Responsive Design**: Adapts seamlessly to mobile, tablet, and desktop screens.
- **Async Submission**: Real-time feedback with `fetch` for a smooth user experience.

## 🛠 Technologies
- **HTML5**: Semantic structure.
- **CSS3**: Advanced layouts with Flexbox and Grid.
- **JavaScript (ES Modules)**: Modern client-side logic.
- **Google Fonts**: [Pretendard](https://github.com/orioncactus/pretendard) for optimal readability in Korean.

## 📁 Project Structure
- `index.html`: Main page structure.
- `style.css`: Modern styling using CSS variables and OKLCH.
- `main.js`: Form handling logic.
- `blueprint.md`: Detailed project design documentation.

## 📝 Usage
1. Open `index.html` in any modern browser.
2. Fill out the form.
3. Your inquiry will be sent directly via the Formspree endpoint (`https://formspree.io/f/mqewanqn`).
